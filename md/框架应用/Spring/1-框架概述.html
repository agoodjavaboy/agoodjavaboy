# 1-жЎҶжһ¶жҰӮиҝ°

  1. SpringжҳҜдёҖдёӘиҪ»йҮҸзә§ејҖжәҗJavaEEжЎҶжһ¶пјҢSpringзҡ„иҪ»йҮҸзә§дҪ“зҺ°еңЁjarиҫғе°‘пјҢе№¶ж•ҙдҪ“дҫқиө–иҫғе°ҸпјҢеҜ№е…¶д»–жЎҶжһ¶зҡ„дҫқиө–д№ҹжӣҙе°‘пјӣ

  2. SpringеҸҜд»Ҙз®ҖеҢ–дјҒдёҡеә”з”ЁејҖеҸ‘зҡ„еӨҚжқӮжҖ§пјӣ

  3. Springзҡ„дёӨдёӘж ёеҝғйғЁеҲҶпјҡIOCжҺ§еҲ¶еҸҚиҪ¬гҖҒAOPйқўеҗ‘еҲҮйқўзј–зЁӢпјҡ

    1. IOCжҺ§еҲ¶еҸҚиҪ¬иЎЁзӨәе°ҶеҲӣе»әеҜ№иұЎзҡ„иҝҮзЁӢдәӨз”ұSpringиҝӣиЎҢз®ЎзҗҶпјҢиҖҢе№¶йқһжүӢеҠЁnewеҲӣе»әгҖӮ
    2. AOPйқўеҗ‘еҲҮйқўеҸҜд»ҘеңЁдёҚж”№еҸҳжәҗд»Јз Ғзҡ„жғ…еҶөдёӢпјҢеҜ№е·ІжңүзЁӢеәҸиҝӣиЎҢеҠҹиғҪеўһејәгҖӮ

  4. SpringжЁЎеқ—з»„жҲҗйғЁеҲҶпјҡ

    еҹәзЎҖжЁЎеқ—пјҡTestпјҲжөӢиҜ•жЁЎеқ—пјүгҖҒBeanпјҲе®һдҫӢз®ЎзҗҶпјүгҖҒCoreпјҲж ёеҝғжЁЎеқ—пјүгҖҒContextпјҲеә”з”ЁдёҠдёӢж–ҮжЁЎеқ—пјүгҖҒExpressionпјҲSpringELиЎЁиҫҫејҸпјүгҖӮ

# Springзү№зӮ№

  1. ж–№дҫҝи§ЈиҖҰпјҢз®ҖеҢ–ејҖеҸ‘пјҡеңЁAOPеҸӮдёҺд№ӢеҗҺпјҢеҠҹиғҪй—ҙзҡ„иҖҰеҗҲеәҰе°ҶйҷҚдҪҺпјӣе№¶дё”еңЁIOCеҸӮдёҺд№ӢеҗҺпјҢеҲӣе»әеҜ№иұЎд»ҘеҸҠжһ„е»әеҜ№иұЎзҡ„иҝҮзЁӢдәӨз”ұSpringпјҢз®ҖеҢ–дәҶејҖеҸ‘гҖӮ
  2. AOPзј–зЁӢж”ҜжҢҒпјҡеңЁдёҚжӣҙж”№жәҗд»Јз Ғзҡ„жғ…еҶөдёӢе°ұеҸҜд»ҘеўһејәеҠҹиғҪгҖӮ
  3. ж–№дҫҝзЁӢеәҸзҡ„жөӢиҜ•пјҡеңЁSpringдёӯйӣҶжҲҗJunitд№ӢеҗҺпјҢеҸҜд»ҘиҝӣиЎҢеҚ•жЁЎеқ—зҡ„еҚ•е…ғжөӢиҜ•пјҢж— йңҖж•ҙдҪ“жөӢиҜ•пјӣ
  4. ж–№дҫҝйӣҶжҲҗе…¶д»–жЎҶжһ¶пјҡеңЁSpringйЎ№зӣ®дёӯпјҢеҸҜд»ҘиҪ»жқҫйӣҶжҲҗе…¶д»–еҠҹиғҪзҡ„жЎҶжһ¶пјӣ
  5. йҷҚдҪҺJavaAPIзҡ„йҡҫеәҰпјҡе°Ҷдј з»ҹJavaAPIиҝӣиЎҢдәҶе°ҒиЈ…пјҢжҡҙйңІжӣҙз®Җдҫҝзҡ„ж“ҚдҪңжҺҘеҸЈпјӣ
  6. еЈ°жҳҺејҸдәӢеҠЎзҡ„ж”ҜжҢҒпјҡдәӢеҠЎзҡ„жҺ§еҲ¶еҸӘйңҖеЈ°жҳҺејҸзҡ„жҺ§еҲ¶пјҢж— йңҖиҖҰеҗҲеҲ°д»Јз Ғдёӯпјӣ
  7. жәҗз Ғз»Ҹе…ёпјҡд»Һи®ҫи®ЎгҖҒз»“жһ„гҖҒжҖқжғіж–№йқўйғҪжҳҜз»Ҹе…ёпјӣ
